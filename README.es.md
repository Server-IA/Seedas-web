# SEEDAS

## Plataforma P2P de Transporte Rural Sostenible

SEEDAS es una plataforma web full-stack responsive que conecta productores rurales con transportadores rurales mediante una economía colaborativa. Permite la publicación de vehículos, necesidades de transporte y rutas comunitarias, facilitando la gestión logística de cargas agrícolas y pecuarias en contextos rurales.

El proyecto surge para resolver la falta de opciones de transporte adaptadas a cargas específicas y vías terciarias, utilizando un enfoque peer-to-peer (P2P), una arquitectura moderna basada en la nube y un sistema modular y escalable.

# Sitio web
https://seedas-web.vercel.app/

# Características Principales

Autenticación y gestión de usuarios con Clerk

Publicación y gestión de servicios de transporte

Estados dinámicos de solicitudes (pendiente, confirmado, en camino, cancelado, finalizado)

Suscripción a transporte comunitario con control de asientos

Visualización de rutas y geocodificación con Mapbox

Notificaciones por cambios de estado

Arquitectura modular y escalable

Backend serverless con Firebase

# Arquitectura y Diseño

Arquitectura full-stack moderna basada en Next.js

Separación por módulos y componentes reutilizables
